# Python-Coded Trading Strategies

> Teaching myself algorithmic trading through Python — one strategy at a time.

This repository is where I learn quantitative finance by building. Each project is a trading strategy coded from scratch, with a detailed learning journal (`LEARNINGS.md`) that documents what I built, what I understood, and what I still need to figure out.

---------

## Why This Exists

I wanted to understand how algorithmic trading actually works — not from textbooks, but by writing the code myself. Every script here is an exercise in breaking down a financial concept into logic, data, and math.

------------

## How Each Project Is Structured

Every project folder contains:
- **CODE** — the Python script
- **LEARNINGS.md** — a journal covering the strategy logic, key concepts learned, limitations I identified, and what I plan to build next

I write the journal after the code runs, not before. It captures what I actually understood, not what I copied.

----------

## Tools

`Python` · `pandas` · `numpy` · `matplotlib` · `yfinance`

--------


## Disclaimer

Personal learning project — not investment advice. Strategies here are educational exercises with known simplifications documented in each project's journal.
